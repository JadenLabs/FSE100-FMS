# FSE 100 - Fine Motor Skills - Final Project

---

## Prerequisites
- Visual Studio Code
- Git CLI

## Installation

1. [Download git](https://git-scm.com/install/windows) on your local system
2. Clone the repository:
    ```bash
    git clone https://github.com/jadenlabs/FSE100-FMS
    ```
3. Change directories into the repo
    ```bash
    cd FSE100-FMS
    ```
4. Open the project in VS Code
    ```bash
    code .
    ```

## Usage

Depending on what you have installed on your system, you can following methods:

**Option 1 - VSC Liveserver:** *(recommended)*

1. Install the **liveserver** plugin for VS Code
2. Right click on `index.html` in the files and click `Open with Live Server`

**Option 2 - Node:**

```bash
npm install -g live-server
cd src
live-server
```

**Option 3 - Python:**

```bash
cd src
python -m http.server 8080
```
